# 科迈 RAS系统 Cookie验证越权漏洞

## 漏洞描述

科迈 RAS系统 存在Cookie验证越权，当 RAS_Admin_UserInfo_UserName 设置为 admin 时可访问后台

## 漏洞影响

<a-checkbox checked>科迈 RAS系统</a-checkbox></br>

## 网络测绘

<a-checkbox checked>app="科迈-RAS系统"</a-checkbox></br>
