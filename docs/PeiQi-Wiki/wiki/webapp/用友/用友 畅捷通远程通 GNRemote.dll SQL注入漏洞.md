# 用友 畅捷通远程通 GNRemote.dll SQL注入漏洞

## 漏洞描述

用友 畅捷通远程通 GNRemote.dll SQL注入漏洞，攻击者通过SQL注入可以获取服务器敏感信息或者使用万能密码登录设备
