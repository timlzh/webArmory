# 众望网络 微议管理系统 后台updatefile.html 任意文件上传漏洞

## 漏洞描述

众望网络 微议管理系统 后台updatefile.html页面任意文件上传漏洞，攻击者通过漏洞可以上传恶意文件控制服务器

## 漏洞影响

<a-checkbox checked>众望网络 微议管理系统</a-checkbox></br>

## 网络测绘
