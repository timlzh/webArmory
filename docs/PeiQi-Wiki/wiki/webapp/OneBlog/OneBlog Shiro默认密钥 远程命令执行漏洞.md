# OneBlog Shiro默认密钥 远程命令执行漏洞

## 漏洞描述

OneBlog 小于v2.2.1 由于使用含有漏洞版本的Apache Shiro和默认的密钥导致存在远程命令执行漏洞

## 漏洞影响

<a-checkbox checked>OneBlog <= v2.2.1</a-checkbox></br>

## 网络测绘

<a-checkbox checked>app="OneBlog开源博客后台管理系统"</a-checkbox></br>
