# 紫光档案管理系统 editPass.html SQL注入漏洞 CNVD-2021-41638

## 漏洞描述

紫光软件系统有限公司（以下简称“紫光软件”）是中国领先的行业解决方案和IT服务提供商。
紫光电子档案管理系统存在SQL注入漏洞。攻击者可利用漏洞获取数据库敏感信息。

## 漏洞影响

<a-checkbox checked>紫光电子档案管理系统</a-checkbox></br>

## 网络测绘
