# EasyImage manager.php 后台任意文件上传漏洞

## 漏洞描述

EasyImage manager.php 存在任意文件上传漏洞，攻击者通过漏洞可以上传恶意文件到服务器获取服务器权限

## 漏洞影响

<a-checkbox checked>EasyImage</a-checkbox></br>

## 网络测绘

<a-checkbox checked>app="EasyImage-简单图床"</a-checkbox></br>
