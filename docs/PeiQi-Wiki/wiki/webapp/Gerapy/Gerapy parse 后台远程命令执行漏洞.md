# Gerapy parse 后台远程命令执行漏洞

## 漏洞描述

Gerapy gerapy/server/core/views.py 中的方法存在任意命令执行，攻击者登录后台后发送特定的请求包即可利用漏洞 

## 漏洞影响

<a-checkbox checked>Gerapy <= 0.9.7</a-checkbox></br>

## 网络测绘

<a-checkbox checked>title="Gerapy"</a-checkbox></br>
