# DocCMS keyword SQL注入漏洞

## 漏洞描述

DocCMS keyword参数存在 SQL注入漏洞，攻击者通过漏洞可以获取数据库信息

## 漏洞影响

<a-checkbox checked>DocCMS</a-checkbox></br>

## 网络测绘

<a-checkbox checked>app="Doccms"</a-checkbox></br>
