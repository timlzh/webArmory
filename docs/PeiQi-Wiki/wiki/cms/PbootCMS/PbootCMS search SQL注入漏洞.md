# PbootCMS search SQL注入漏洞

## 漏洞描述

PbootCMS 搜索模块存在SQL注入漏洞。通过漏洞可获取数据库敏感信息

## 漏洞影响

<a-checkbox checked>PbootCMS < 1.2.1</a-checkbox></br>

## 网络测绘 

<a-checkbox checked>app="PBOOTCMS"</a-checkbox></br>
