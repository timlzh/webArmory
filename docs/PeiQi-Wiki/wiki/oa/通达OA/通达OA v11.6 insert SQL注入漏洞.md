# 通达OA v11.6 insert SQL注入漏洞

## 漏洞描述

通达OA v11.6 insert参数包含SQL注入漏洞，攻击者通过漏洞可获取数据库敏感信息

## 漏洞影响

<a-checkbox checked>通达OA v11.6 </a-checkbox></br>

## 网络测绘

<a-checkbox checked>app="TDXK-通达OA" </a-checkbox></br>
