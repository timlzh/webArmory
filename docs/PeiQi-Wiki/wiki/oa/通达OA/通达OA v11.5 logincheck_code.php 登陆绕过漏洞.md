# 通达OA v11.5 logincheck_code.php 登陆绕过漏洞

## 漏洞描述

通达OA v11.8 logincheck_code.php存在登陆绕过漏洞，通过漏洞攻击者可以登陆系统管理员后台

## 漏洞影响

<a-checkbox checked>通达OA v11.8</a-checkbox></br>

## 网络测绘

<a-checkbox checked>app="TDXK-通达OA" </a-checkbox></br>
