# Apache Hadoop Yarn RPC 远程命令执行漏洞

## 漏洞描述

 Hadoop Yarn RPC未授权访问漏洞存在于Hadoop Yarn中负责资源管理和任务调度的ResourceManager，成因是该组件为用户提供的RPC服务默认情况下无需认证即可访问，
