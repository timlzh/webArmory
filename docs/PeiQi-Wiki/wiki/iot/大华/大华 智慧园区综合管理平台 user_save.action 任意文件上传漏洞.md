# 大华 智慧园区综合管理平台 user_save.action 任意文件上传漏洞

## 漏洞描述

大华 智慧园区综合管理平台存在未授权访问漏洞，攻击者通过构造特殊的请求包可以创建新用户，再利用文件上传漏洞获取服务器权限

## 漏洞影响

<a-checkbox checked>大华 智慧园区综合管理平台</a-checkbox></br>

## 网络测绘

<a-checkbox checked>app="dahua-智慧园区综合管理平台"</a-checkbox></br>
