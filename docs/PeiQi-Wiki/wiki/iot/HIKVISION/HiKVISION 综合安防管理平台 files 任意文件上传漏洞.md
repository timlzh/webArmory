# HiKVISION 综合安防管理平台 files 任意文件上传漏洞

## 漏洞描述

HiKVISION 综合安防管理平台 files 接口存在任意文件上传漏洞，攻击者通过漏洞可以上传任意文件

<a-checkbox checked>HiKVISION 综合安防管理平台</a-checkbox></br>
