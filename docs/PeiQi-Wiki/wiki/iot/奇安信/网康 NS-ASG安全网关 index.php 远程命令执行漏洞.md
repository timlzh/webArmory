# 网康 NS-ASG安全网关 index.php 远程命令执行漏洞

## 漏洞描述

网康 NS-ASG安全网关 index.php文件存在远程命令执行漏洞，攻击者通过构造特殊的请求包可以获取服务器权限

## 漏洞影响

<a-checkbox checked>网康 NS-ASG安全网关</a-checkbox></br>

## 网络测绘

<a-checkbox checked> title=="网康 NS-ASG 应用安全网关"</a-checkbox></br>
