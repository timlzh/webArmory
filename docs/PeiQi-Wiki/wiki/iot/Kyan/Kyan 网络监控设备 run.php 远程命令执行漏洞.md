# Kyan 网络监控设备 run.php 远程命令执行漏洞

## 漏洞描述

Kyan 网络监控设备 run.php可在身份验证的情况下执行任意命令, 配合账号密码泄露漏洞，可以获取服务器权限，存在远程命令执行漏洞

## 漏洞影响

<a-checkbox checked>Kyan</a-checkbox></br>
