# 频谱隐写

## 关于

音频中的频谱隐写是将字符串隐藏在频谱中，当音频中包含一段杂音或刺耳的声音时，就有可能通过频谱隐写来进行信息的隐藏。对于这类题目，通过各类工具就可以获取到需要的信息。
