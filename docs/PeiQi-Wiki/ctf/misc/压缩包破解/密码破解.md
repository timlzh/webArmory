# 密码破解

## 关于

ZIP,RAR,7Z等后缀为压缩包格式后缀，用户可以通过压缩的方法将多个目录和文件压缩至一个 ZIP或RAR 文件中，并附加密码来保证信息的安全，但弱口令将会导致压缩包密码被爆破的问题。

目前已经有了完善的工具进行常见几类的压缩包密码破解，当密码只存在数字或弱口令时，通过一定的时间可以破解出压缩包的密码获取内容
