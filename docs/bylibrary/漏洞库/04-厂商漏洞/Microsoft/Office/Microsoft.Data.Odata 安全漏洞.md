##### **|**漏洞详情                        

Microsoft.Data.Odata是美国微软（Microsoft）公司发布的一款支持开放数据协议（OData）的.NET  Framework序列化库。 Microsoft.Data.Odata中存在拒绝服务漏洞，该漏洞源于程序没有正确的处理Web请求。远程攻击者可通过向Odata应用程序发送特制的请求利用该漏洞使Odata Web应用程序无法正常服务。

  end
