# Readme
部分脚本文件使用说明，详细使用分析请参考[vulnerability-list](https://github.com/zhzyker/exphub/tree/master/struts2#vulnerability-list)中的[使用]
- VER: 漏洞影响版本，一般情况下不在影响范围的版本没有相关漏洞
- USE: 脚本文件使用说明，大部分写在了脚本里，执行即可见
- EXP: 脚本利用示例，以及执行效果
- DES: 部分特殊脚本文件的特殊描述
